## 五、彩种切换数据流

```
page.tsx（服务端组件）
  ├── fetchAllLegacyModulesByGame()
  │     ├── fetchAllLegacyModules([], 3)  → 台湾彩 modules[]
  │     ├── fetchAllLegacyModules([], 2)  → 澳门彩 modules[]
  │     └── fetchAllLegacyModules([], 1)  → 香港彩 modules[]
  │
  ├── mergeSiteAndLegacy()  →  modulesByGame = { taiwan, macau, hongkong }
  │
  └── transformSitePageData() → LotteryPageData { modulesByGame }
                                      ↓
HomePageClient（客户端组件）
  ├── useState<LotteryGame>("taiwan")
  ├── onGameChange → 切换 activeGame
  │
  ├── PreResultBlocks
  │     └─ modulesByGame[activeGame] → 3个核心模块数据
  │
  └── PredictionModules
        └─ modulesByGame[activeGame] → 其余模块数据
```

**关键点**：
- `modulesByGame` 是 `Record<LotteryGame, PublicModule[]>`，已在 `page.tsx` 中预加载所有三种彩种数据
- 切换彩种时无额外网络请求，直接从内存切换数据
- 每个自定义渲染器接收 `PublicModule`，其 `history` 数组已是对应彩种的数据
